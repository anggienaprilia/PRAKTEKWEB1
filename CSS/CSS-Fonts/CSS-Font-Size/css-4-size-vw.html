Ubah ukuran jendela browser untuk melihat skala ukuran teks.

Gunakan unit "vw" saat mengukur teks. 10vw akan mengatur ukurannya menjadi 10% dari lebar viewport.

Viewport adalah ukuran jendela browser. 1vw = 1% dari lebar area pandang. Jika lebar area pandang 50cm, 1vw adalah 0,5cm.
